Uber Ride Analytics Dashboard

Built using Power BI for interactive data visualization.
Tracks key metrics such as total bookings, revenue, trip distance, and ride duration.
Analyzes peak booking hours, ride trends, and location performance.
Includes dynamic filters, KPIs, and drill-through functionality for deeper insights.
